# Hotel Management System (C++ OOP)

A console-based **Hotel Management System** developed in **C++** using **Object-Oriented Programming (OOP)** principles.  
This project allows both **customers** and **admin** to manage hotel rooms, bookings, and revenue. It also features **file handling** to save and load data.

---

## Features

### Customer Features
- View all available rooms.
- Book a room with personal details and stay duration.
- Checkout from a room.
- View personal booking details.

### Admin Features
- Admin login with password protection.
- View all rooms and bookings.
- Add or remove rooms dynamically.
- Update room prices for different types.
- View total revenue generated by the hotel.

## File Handling
- Saves hotel room data and bookings in a **`rooms.txt`** file.
- Loads room data automatically when the program starts.
- Ensures that bookings and updates persist between program runs.

---

## Technologies Used
- **C++** for programming.
- **OOP Concepts**: Classes, Inheritance, Encapsulation.
- **File Handling** for persistent data storage.
- **Console I/O** with formatted output (`iomanip`).
